Privacy and Endpoints

JSON endpoints only render items that are not set to "private." If you would like to change the privacy setting for an item you created, you may do so at any time by navigating to your item's page and clicking "Edit Item".


Other Endpoints

Endpoints also exist for all non-private items. The URLs for these endpoints use the following format:

/item/<string:item_name>/JSON

Example:

/item/Visor/JSON
